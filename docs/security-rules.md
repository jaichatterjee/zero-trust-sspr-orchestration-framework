# Security Rules

## Non-Negotiable Rules

### Rule 1

Power Automate orchestrates workflow only.

Power Automate cannot make trust decisions.

---

### Rule 2

Device trust validation is required for automated recovery.

---

### Rule 3

MFA reset requires device neutralization.

Examples:

- Device wipe
- Device block

---

### Rule 4

Critical risk scenarios cannot be automated.

---

### Rule 5

All actions must be logged and auditable.
