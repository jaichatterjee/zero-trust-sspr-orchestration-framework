# Sample User Flow

Scenario:

User forgot password while using a managed device.

---
``` text
Flow:

User clicks desktop shortcut

↓  

Power Automate opens recovery interface

↓  

User selects:

Forgot Password

↓  

System validates:

- Device compliance
- Identity risk
- Session context

↓  

Risk evaluation result:

Medium

↓  

Action:

Enforce MFA

↓  

Password reset allowed
```
